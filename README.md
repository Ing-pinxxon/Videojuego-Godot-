# Videojuego Tipo Zelda - Godot

Un proyecto de práctica aplicada de videojuego tipo Zelda desarrollado en **Godot Engine** usando **GDScript**.

## 📋 Descripción

Este proyecto es un juego 2D de aventura inspirado en la saga Zelda, donde el jugador deberá enfrentarse a enemigos, resolver acertijos, avanzar por diferentes niveles y gestionar la vida del personaje principal.

## 🎮 Características

- **Combate**: Sistema de batallas contra enemigos
- **Niveles**: Múltiples niveles para explorar
- **Sistema de Vida**: Gestión de puntos de vida del jugador
- **Acertijos**: Puzzles para resolver y avanzar en el juego
- **Exploración**: Mundo interactivo para descubrir

## 🛠️ Tecnología

- **Engine**: Godot Engine
- **Lenguaje**: GDScript
- **Plataforma**: Multiplataforma (Windows, Linux, macOS)

## 📦 Requisitos

- Godot Engine 4.x (o la versión utilizada en el proyecto)
- Un editor de código (recomendado VSCode con extensión Godot)

## 🚀 Cómo Ejecutar

1. Clona el repositorio:
   ```bash
   git clone https://github.com/Ing-pinxxon/Videojuego-Godot-.git
   ```

2. Abre el proyecto en Godot Engine

3. Presiona el botón Play o usa `F5` para ejecutar el juego

## 📁 Estructura del Proyecto

```
Videojuego-Godot-/
├── scenes/          # Escenas del juego
├── scripts/         # Scripts en GDScript
├── assets/          # Gráficos, sonidos y recursos
├── levels/          # Archivos de niveles
└── project.godot    # Archivo de configuración del proyecto
```

## 🎯 Objetivos del Jugador

- Avanzar a través de los niveles
- Derrotar enemigos en combate
- Resolver acertijos para acceder a nuevas áreas
- Mantener la vida del personaje
